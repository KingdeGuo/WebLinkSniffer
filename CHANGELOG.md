# 更新日志

本文档记录「网址获取」浏览器插件的所有重要变更。

## [1.3.0] - 2026-05-03

### 新增
- 🚫 链接右侧一键屏蔽按钮：单条链接直接加入屏蔽列表，无需复制粘贴到后台
- 🌐 链接右侧一键屏蔽域名按钮：一键屏蔽链接所属的整个域名
- ☑️ 多选批量操作工具栏：勾选多个链接后，支持批量屏蔽链接、批量屏蔽域名、批量复制、批量打开
- 📋 快速复制功能：选中链接后可一键复制所有选中链接的 URL 及标题
- 🔄 撤销功能：每次屏蔽/域名屏蔽后 Toast 通知带有「撤销」按钮
- 🏷️ 已屏蔽链接快捷管理：底部展示已屏蔽链接标签，点击可撤销，支持一键清空

### 优化
- 域名过滤改为始终生效（不受 options 页面开关控制），popup 一键屏蔽的域名即刻过滤
- 页面搜索功能优化
- UI 交互体验大幅提升

## [1.2.0] - 2026-05-03

### 新增
- 具体 URL 屏蔽功能：支持精确匹配屏蔽特定链接
- 一键屏蔽按钮：链接右侧可快速屏蔽当前链接

## [1.1.1] - 2026-05-03

### 修复
- 修复去重漏判：剥离评论回复参数（`replyTo`）、营销追踪参数（`utm_*`、`fbclid` 等）、时间戳/缓存参数等不影响页面内容的查询参数后再做去重匹配
- 修复去重漏判：同一页面使用 `http` 和 `https` 不同协议或 `www.` 前缀差异时，统一规范化后再去重

## [1.1.0] - 2026-05-03

### 新增
- 链接智能排序功能：按链接质量评分排序，已打开链接自动移到末尾
- 链接去重功能：自动合并含 `#` 锚点及相同路径去重的链接
- 域名过滤功能：通过配置页面设置要过滤的域名列表
- 关键词过滤功能：支持按链接文本关键词过滤链接
- 配置页面（options.html / options.js）
- 隐藏已打开链接的选项
- 插件图标（16/48/128 像素）

### 优化
- 链接排序算法：综合域名匹配度、页面位置、标题质量、URL 清洁度等多维度评分
- 获取链接失败时的自动重试机制
- 项目目录规范化：测试文件移至 `tests/`，开发文档移至 `docs/`

### 仓库
- 添加 `.gitignore`
- 添加 `LICENSE` (MIT)
- 添加 `CHANGELOG.md`

## [1.0.0] - 2024-03-08

### 新增
- 初始版本发布
- 自动获取当前网页中所有超链接
- 分页展示：每页 5 条链接，支持上一页/下一页导航
- 缓存管理：通过 Chrome Storage API 持久化已打开链接
- 批量打开：一键打开当前页所有链接
- 自动将已打开链接移动到末尾
